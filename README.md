# pwsh-profile

## Installation

## Pre-Install

Get PowerShell 7+ from here: https://github.com/PowerShell/PowerShell/releases.

Or check: https://github.com/PowerShell/PowerShell.

## Install

```
git clone https://github.com/shatfel/pwsh-profile
```

## Use

```
cd pwsh-profile

pwsh profile.ps1 [-force]
```
